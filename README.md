# Operations Analytics Platform

## Overview
This project demonstrates an end-to-end operations analytics platform built to analyse
retail and logistics performance using SQL, Python and Power BI.

It simulates a real-world analytics environment, focusing on automated reporting,
trend analysis and executive-level decision support.

## Tools & Technologies
- SQL – automated KPI reporting pipelines
- Python – time-series trend and trend variance analysis
- Power BI – interactive dashboards for operational and commercial reporting
- ETL workflows – data cleaning, standardisation and validation
- AI-assisted analytics – SQL drafting, documentation and insight summarisation (human-validated)

## Key Capabilities
- Automated weekly KPI reporting covering revenue, order volume and on-time delivery
- Python-based seasonality and performance trend analysis
- Executive-ready dashboards for operational decision-making
- Multi-source data transformation and validation
- Responsible use of AI tools to accelerate analytics workflows

## Repository Structure
- `sql/` – KPI queries and automated reporting logic
- `python/` – trend analysis and forecasting notebooks
- `powerbi/` – dashboard screenshots and descriptions (to be added)

## Business Value
- Faster and more consistent reporting
- Clear visibility of operational performance drivers
- Decision-ready insights for stakeholders
